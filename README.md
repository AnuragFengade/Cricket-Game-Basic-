🏏 Cricket Champion League (Java Project)

A console-based Cricket League Simulation System built in Java that models a full tournament with 10 custom teams, league matches, and knockout rounds — just like a mini IPL.

🚀 Features

✅ 10 Custom Teams
✅ Automatic Match Simulation
✅ Random Score Generation
✅ Round-Robin League System
✅ Points Table with Sorting
✅ Semi-Finals & Final Match
✅ Champion Declaration 🏆
✅ Clean OOP-based Design

⚙️ How It Works
Teams Initialization
10 teams are created with 11 players each
League Matches (Round Robin)
Each team plays against every other team
Match Simulation
Random scores generated
Winner gets 2 points
Points Table
Teams sorted based on points
Knockout Stage
Top 4 teams qualify
Semi-finals → Final
Champion Declared 🏆
▶️ How to Run
